<p align="center">Mobile app for controlling your PC cursor from Android/iOS</p>

## Quick Start

**On PC**
>1. Download and run the server binary from the [Releases page](https://github.com/KMRH47/pointZ-new/releases/latest) (runs in system tray)
>2. Right-click the tray icon -> "Show QR Code"
>3. Scan the QR code with your phone to download the mobile app
>4. The app will auto-discover and connect to your server

## Downloads
